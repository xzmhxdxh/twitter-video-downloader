# SaveXTweet - Minimalist Twitter/X Video Downloader

[![Website Status](https://img.shields.io/website?url=https%3A%2F%2Fsavextweet.com&up_message=Online&up_color=success&style=flat-square)](https://savextweet.com)

**[SaveXTweet.com](https://savextweet.com)** is a fast, reliable, and completely free web application designed to download videos, GIFs, and images from X (formerly Twitter) in the highest available quality. 

This repository serves as the official documentation, feature request hub, and issue tracker for the SaveXTweet web tool.

## 🚀 Live Demo & Tool
**Access the tool directly here: [https://savextweet.com](https://savextweet.com)**

## ✨ Why I Built This (The "Less is More" Philosophy)

The internet is currently flooded with Twitter video downloaders that are bloated with spammy pop-ups, misleading ad buttons, and clunky interfaces. 

As a web developer, I believe in the **"Less is More"** design philosophy. I built SaveXTweet to solve my own frustration, focusing on:
* **Zero friction:** No registration, no app installations.
* **Clean UI:** A modern, frosted-glass inspired dark mode that is easy on the eyes.
* **Pure utility:** Paste the link, click download. That's it.

## ⚡ Key Features

- **Download Twitter/X Videos:** Save MP4 videos in original resolutions (1080p, 720p, etc.).
- **Save Twitter GIFs:** Native support for extracting and saving GIFs from any tweet.
- **Video to MP3:** Easily convert and extract the audio track from any Twitter video.
- **Cross-Platform:** Works perfectly on PC, Mac, iOS (Safari), and Android browsers.
- **Privacy First:** We do not track your download history or store the media on our servers.

## 🛠️ How to Use

1. **Copy the link:** Go to X/Twitter, find the post with the video/GIF, and copy its shareable link.
2. **Paste the URL:** Visit **[SaveXTweet](https://savextweet.com)** and paste the link into the main input field.
3. **Download:** Click the download button and choose your preferred format (MP4 or MP3).
